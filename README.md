# Install
1. `git clone https://github.com/DaddyBigFish/enumx.git .git/enumx`
2. `cd .git/enumx`
3. `chmod +x enumx`
4. `sudo mv enumx /usr/local/bin`
5. `enumx <TARGET> | tee enumx-output.txt`
